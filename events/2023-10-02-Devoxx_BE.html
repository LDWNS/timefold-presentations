Getting started

Start coding today

  1. Go to timefold.ai
  2. Click the Quickstarts repo button.
  3. Pick a quickstart from the README.
  4. Run it:
$ git clone https://github.com/TimefoldAI/timefold-quickstarts.git
...
$ cd timefold-quickstarts/use-cases/vehicle-routing-capacity
$ mvn quarkus:dev
...
